This repository can be read in MakeCode.

* open [https://makecode.microbit.org/beta](https://makecode.microbit.org/beta)
* click on **Import**
* click on **Import URL**
* Input **https://github.com/asondemita/contact-warning-and-tracing** and click on **Go ahead!**
